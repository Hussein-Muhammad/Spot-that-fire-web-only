Spot That Fire !

features list :

• Mobile Application (Android (java)), Web Application (flask-Python-creative design).

• Categorized as an event Based application that will focus on fire accidents and reporting it.
• The application will have users DB, each user will be able to report a fire accident and share it globally on news feed and the report can contain textual, multimedia and geolocation.
• The applications will have a news feed will be a place Where you can see fire reports by users across the world.
• The application will have two news feed (Hall of Flame ) globally and locally based on geolocation.
• The application will notify people around X distance from the location of the fire.
• The application will verify the reports based on the data by a fancy machine learning algorithm which will filter spamming, non-related to fire accidents.
• The application will have a guide tab to show instruction about how to put down fire, first aids for level one, two and third burning, the quick number of fire fighting in the area.
• The application will have a button ask for help to call firefighting number in local area and show the instructions to avoid fire.
• Display the weather conditions, local traffic around the fire area.
